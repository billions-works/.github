# Billions

Billions is a regulated stablecoin payment gateway for businesses that want to accept stablecoin payments and settle in fiat.

We help merchants launch stablecoin payments without rebuilding their operations from scratch. Customers can pay in stablecoins, while businesses keep familiar payout and settlement flows.

## What we build

- Stablecoin checkout for web and mobile products
- Hosted payment pages and invoice flows
- Merchant API for payment creation and status tracking
- Internal payout and settlement infrastructure
- Developer tooling for faster integrations

## Developer resources

- Website: [billions.works](https://billions.works)
- TypeScript SDK: [billions-works/billions-sdk-ts](https://github.com/billions-works/billions-sdk-ts)

## Integration

If you want to integrate Billions into your product, start with the TypeScript SDK.
